The Finnish repositories use a "dot" extended form of DC 1.1. This quick analysis compares that vocabulary to SRAP.

* The file [finnDC.txt](finnDC.txt) is a list of the DC 1.1 "extended" elements in the Finnish repo vocabulary. 
* The file [finnVsrap.csv](finnVsrap.csv) is a table of the SRAP elements, the Finnish repo elements, and some notes where the "dot notation" used on DC 1.1 could translate to a DC Terms element. (Not many)
